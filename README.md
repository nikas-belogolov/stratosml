
# STRATOSML - C++ Machine Learning Library


## How to compile
```bash
make
g++ -std=c++20 -I./include -L./lib -fcompare-debug-second <path to file>.cpp -o <output file> -larmadillo -lblas -llapack -lstratosml

```
